Project Working Flow

Use this section while explaining your project in review or viva.

1

Image Collection

Farmer captures the affected leaf using mobile camera or uploads existing field image.

2

Image Preprocessing

The system resizes, cleans, normalizes, and prepares the image for machine learning analysis.

3

Feature Extraction

Color, texture, spots, leaf patterns, and infected regions are observed by image processing or CNN layers.

4

Disease Classification

The trained ML/CNN model predicts whether the leaf is healthy or affected by a particular disease.

5

Recommendation

The website displays disease name, confidence, symptoms, prevention, and treatment suggestions.
